Rajce album downloader

A browser-extension equivalent of the U++ desktop app — download images and videos from rajce.net albums, including password-protected ones.

Why an extension and not a plain web page?

rajce.net does not send CORS headers, so a page hosted on GitHub Pages cannot fetch album pages or media directly. A browser extension is granted cross-origin access to rajce.net via its manifest host_permissions, so it works without any proxy and keeps your credentials on your machine.

Download & install (Chrome / Edge / Brave)

Download latest build (.zip)

  1. Click the button above to download the latest packaged build.
  2. Unzip it to a folder of your choice.
  3. Open chrome://extensions (or edge://extensions).
  4. Enable Developer mode.
  5. Click Load unpacked and select the unzipped folder.
  6. Click the toolbar icon to open the downloader.

Chrome blocks one-click installs of .crx files from outside the Chrome Web Store, so the build is shipped as a .zip you load unpacked. See app-web/README.md for details and limitations.

Usage

  1. Paste an album URL (e.g. https://user.rajce.net/album).
  2. For protected albums, tick Enable album authorization and fill user + password.
  3. Choose how to save: into your Downloads folder, or as a single ZIP.
  4. Click Download.
